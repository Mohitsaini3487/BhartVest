# **App Name**: BharatVest

## Core Features:

- AI-Powered Market Analysis: Provides AI-driven insights and analytics on the Indian stock market, including NIFTY and SENSEX, using natural language queries in English and Hindi. It recommends trading strategies for Indian markets using a provided Gemini API tool.
- Real-time Data Integration: Fetches live stock prices from NSE/BSE using free APIs and integrates data into the application. Supports real-time data sync with Excel files and includes historical data for analysis. It will track NSE/BSE trading hours.
- Portfolio Management: Enables users to simulate trades, manage portfolios, track profit and loss, and create personalized watchlists. It displays data for popular Indian stocks like Reliance and TCS.
- Excel Integration: Allows users to import and export portfolio data via Excel files, ensuring seamless integration with existing workflows. All data should be available and downloadable to excel sheets.
- Personalized Dashboard: Offers a customizable dashboard with key metrics, performance analytics, and AI-driven investment recommendations. Visualized using charts and real-time datasets to offer an interactive analytics tool
- Multi-Language Support: Provides a user interface in both English and Hindi, catering to a wider audience in India. UI Components such as headings, paragraphs, labels should all be available in Hindi.
- Mobile-Responsive Design: Features a responsive design that adapts to different screen sizes, ensuring a seamless experience for mobile users.

## Style Guidelines:

- Primary color: Deep saffron (#FF9933), reflecting the vibrancy and optimism of the Indian market.
- Background color: Light beige (#F5F5DC), offering a neutral backdrop to highlight data visualization and key insights.
- Accent color: Teal (#008080), providing contrast for calls to action and real-time trading elements.
- Body font: 'PT Sans' for clear readability and comfortable reading of detailed financial information.
- Headline font: 'Space Grotesk' to capture a forward-thinking spirit; when pairing, use Space Grotesk for headlines and 'PT Sans' for body.
- Custom-designed icons representing Indian market trends and financial instruments.
- Clean and intuitive layout with clear navigation for effortless access to information and tools.
- Smooth data transitions for real-time updates, reflecting dynamic market data and portfolio shifts.
